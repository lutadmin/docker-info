**What is Docker and Container ?**

"Docker" is the base-layer where you set/run your containers, "Docker image (ubuntu,linux)" is required to set the environment that Containers can run. Docker technology is a lightweight, standalone, executable package of software to run your Containers.

Container (packaged up programming code, bundle of applications, application libraries) share the **host is- Docker** operating system's kernel and libraries, while providing isolation for its own application code's and its dependencies.


**Docker Containers vs Virtual Machine** 

Containers uses it's HOST- Docker's operating system kernel, libraries which make Container's to work fast and executing it's own programing code.  
On the otherhand Virtual Machine(VMs) require a full-fledged Operating System and compatible Hypervisor to run, which is highly resource-intensive (compute,storage,RAM).
